Black market (two rotator slots)

Reads/writes blackmarket_items (two entries), blackmarket_usedpools, and blackmarket_trackerstate on the same object as your save’s profile: block (or root if those keys are at the top). Item type is the short code (e.g. DAD_SM). Item comp is the full component path (legendary *.comp_05_legendary_* or pearl *.comp_06_pearl_*) — pick from suggestions or paste from a decoder. Used pools is internal vendor rotation state (often starts with a large value such as 2147483648); only change it if you know you need to; leave all three boxes empty when applying to leave pools untouched in YAML.
